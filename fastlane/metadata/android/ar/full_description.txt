Gitling هو عميل Git لنظام Android، مبني باستخدام Jetpack Compose و Material 3.

هو شوكة من MGit (وهو بدوره استمرار لـ SGit)، تم تحديثه بأدوات Android الحديثة، وواجهة Compose مكتوبة من الصفر، وتسجيل دخول إلى GitHub بنقرة واحدة، وهوية جديدة.

الميزات:

* استنساخ وإنشاء وحذف المستودعات المحلية
* اتصال بنقرة واحدة مع GitHub (OAuth Device Flow) - لا حاجة للتعامل مع الرموز أو كلمات المرور يدوياً لمستودعات GitHub
* HTTP/HTTPS/SSH، بما في ذلك SSH مع عبارة مرور للمفتاح الخاص
* توثيق باسم المستخدم/كلمة المرور ورمز الوصول الشخصي لأي مضيف Git
* تصفح الملفات وسجل الالتزامات، وعرض الفروقات (diff) بين الالتزامات
* تسجيل الخروج، وإنشاء، ودمج الفروع والوسوم المحلية/البعيدة
* حالة git، الإعداد (staging)، git rebase، git cherry-pick، git checkout &lt;ملف&gt;
* إنشاء وإدارة المفاتيح الخاصة
* استيراد مستودع موجود من ذاكرة الجهاز
* وضع فاتح/داكن يتبع النظام، مع خيار بين لون العلامة التجارية الثابت أو اللون الديناميكي Material You القائم على الخلفية

Gitling لا يحتوي على محرر نصوص داخلي - لتعديل الملفات، ستحتاج إلى تطبيق محرر مثبت يدعم فتح الملفات عبر آلية File Provider في Android (معظم تطبيقات تحرير النصوص والكود تفعل ذلك).
